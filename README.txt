  Student Performance Analysis

This repository contains code for analyzing the academic performance of high school students based on their demographic characteristics, study habits, and alcohol consumption. The data was collected from two courses: mathematics and Portuguese language. The purpose of the analysis is to identify any correlations between these factors and academic performance, particularly with regard to alcohol consumption.

  Files

alc-data-2023-05-07.csv: Data on math course students including alcohol consumption.
parent-data-2023-05-05.csv: Data on Math course students including parents' education.
student_analysis.ipynb: Jupyter notebook containing Python code for data processing and analysis.

  Requirements

The code in this repository was written in Python 3 and uses the following libraries:

Pandas
NumPy
Matplotlib
CSV

  Usage

To run the code in the Jupyter notebook, you'll need to install Jupyter and the required libraries. You can do this using pip:

  pip install jupyter pandas numpy matplotlib csv

Then, start the Jupyter notebook server by running the following command:

  jupyter notebook

To get the visualizations, you need to run the code with the play button on top.
